A simple Machine Learning + Flask web application that predicts whether a loan application will be approved or rejected based on user input.

The project uses a trained ML model to analyze factors like applicant income, credit history, loan amount, and other personal details. Users can enter their information through a web form, and the system instantly displays the prediction result with a user-friendly interface.

1)Enter Personal Details
Fill the form fields:
Gender → Select Male (1) or Female (0)
Married → Yes (1) or No (0)
Dependents → 0 / 1 / 2 / 3+
Education → Graduate (1) or Not Graduate (0)
Self Employed → Yes (1) or No (0)

2)Enter Financial Details
Applicant Income → Monthly income (e.g., 5000)
Co-applicant Income → Secondary income (e.g., 2000)
Loan Amount → Loan amount (e.g., 150)
Loan Amount Term → Duration (e.g., 360)

3)Enter Credit Information
Credit History
1 → Good (High chance of approval ✅)
0 → Bad (High chance of rejection ❌)

4)Select Property Area
0 → Rural
1 → Semi-Urban
2 → Urban

5)Submit the Form
Click on Submit Button
