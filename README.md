A simple Machine Learning + Flask web application that predicts whether a loan application will be approved or rejected based on user input.

The project uses a trained ML model to analyze factors like applicant income, credit history, loan amount, and other personal details. Users can enter their information through a web form, and the system instantly displays the prediction result with a user-friendly interface.
